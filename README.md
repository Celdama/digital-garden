garden is an opinionated, batteries included boilerplate for md/x authoring with next.js.

## tech

- nextjs
- md/mdx authoring - uses [`next-mdx-remote`](https://github.com/hashicorp/next-mdx-remote)
- theme-ui
- styled-components
- jest/github actions (ci/cd)
- lint/prettier

## run or deploy

fork repo, then

```bash
yarn
yarn dev
```

deploy with vercel
